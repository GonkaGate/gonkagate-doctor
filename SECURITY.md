# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues to:

- Email: `security@gonkagate.com`
- Or via GitHub Security Advisories (preferred, if enabled): `https://github.com/GonkaGate/gonkagate-doctor/security/advisories`

Include as much detail as possible:

- affected versions
- reproduction steps / proof of concept
- impact assessment
- suggested fix (if you have one)

## Supported Versions

Only the latest published version is supported with security updates.
